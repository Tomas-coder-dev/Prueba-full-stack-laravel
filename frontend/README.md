# Prueba Técnica Fullstack - Frontend (SPA)

Este repositorio contiene la parte Frontend de la prueba técnica para el Gestor de Tareas. Es una Single Page Application (SPA) construida sobre una base de Laravel, utilizando Inertia.js, Vue 3 (Composition API), Pinia para el manejo de estado y TailwindCSS.

## Stack Tecnológico
* Node.js >= 16.x y NPM
* Vue 3
* Inertia.js
* Pinia
* TailwindCSS (v3/v4 via Vite)
* Axios

## Pasos de Instalación

1. Clona el repositorio e ingresa a la carpeta `frontend`.
2. Instala las dependencias de Node.js y PHP:
   npm install
   composer install
