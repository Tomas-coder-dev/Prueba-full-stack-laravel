# Prueba Técnica Fullstack - Backend (API REST)

Este repositorio contiene la parte del Backend de la prueba técnica para el Gestor de Tareas. Está desarrollado utilizando Laravel y expone una API RESTful protegida mediante autenticación JWT.

## Stack Tecnológico
* PHP >= 8.1
* Laravel 10+
* MySQL
* Tymon JWT Auth

## Pasos de Instalación

1. Clona el repositorio e ingresa a la carpeta `backend`.
2. Instala las dependencias de PHP:
   composer install
